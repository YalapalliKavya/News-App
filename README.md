# 📰 News App

A simple and interactive **News Application** built using **Python and Flask** that fetches and displays the latest news articles. The app allows users to view current news updates with a clean and responsive interface.

---

## 🚀 Features

* 📰 Fetches latest news articles
* 🔍 Displays news with title and description
* 🎨 Simple and clean UI using HTML & CSS
* ⚡ Fast backend using Flask
* 🔐 Secure API key management using `.env`

---

## 🛠️ Technologies Used

* **Python**
* **Flask**
* **HTML**
* **CSS**
* **REST API**
* **dotenv**

## ⚙️ Installation

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone https://github.com/YalapalliKavya/News-App.git
```

### 2️⃣ Navigate to the project folder

```
cd News-App
```

### 3️⃣ Create virtual environment

```
python -m venv venv
```

### 4️⃣ Activate virtual environment

Windows:

```
venv\Scripts\activate
```

### 5️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 6️⃣ Create `.env` file

```
GROQ_API_KEY=your_api_key_here
```

### 7️⃣ Run the application

```
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
